## Quick Start

### For Citizens

1. Read [Getting Started for Non-Developers](./GETTING-STARTED-NON-DEVELOPERS.md)
2. Watch Video 2: "Your First Visualization"
3. Explore the [Interactive Tutorials](./INTERACTIVE_TUTORIALS.md)

### For Developers

1. Read [Getting Started](./GETTING-STARTED.md)
2. Review [API Reference](./api-reference/overview.md)
3. Watch Video 5: "Developer Quickstart"

### For Journalists

1. Read [Data Journalism Toolkit](./DATA_JOURNALISM_TOOLKIT.md)
2. Watch Video 4: "Data Journalism Tools"
3. Review [Citation Guidelines](./DATA_JOURNALISM_TOOLKIT.md#citation-guidelines)

### For Government Staff

1. Read [Partnership Proposal Template](./PARTNERSHIP_PROPOSAL_TEMPLATE.md)
2. Review [Implementation Support Package](./PARTNERSHIP_PROPOSAL_TEMPLATE.md#implementation-support-package)
3. Contact opendata@ite.gov.rs
