- Large community and resources
- Advanced visualizations

**Why Choose Vizualni Admin Srbije:**

| Aspect                   | Tableau Public                                 | Vizualni Admin Srbije               |

## Migration Guides

### From Tableau to Vizualni

**Migration effort:** 1-2 weeks per dashboard

```typescript
// Tableau calculation
SUM([Budget 2024]) - SUM([Budget 2023])

// Vizualni equivalent
import { calculateChange } from '@vizualni/core';
const change = calculateChange(data, 'budget', 'year', [2023, 2024]);
```

**Key differences:**

- Tableau's calculated fields → TypeScript functions
- Tableau's parameters → Component props
- Tableau's filters → Data transformation before render

### From Power BI to Vizualni

**Migration effort:** 1 week per dashboard

**Key differences:**

- Power BI DAX → TypeScript/JavaScript
- Power Query → data.gov.rs connectors
- Power BI service → Vizualni dashboard or embed
