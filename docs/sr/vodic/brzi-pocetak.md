# Водич за брзи почетак

**Поставите Vizualni Admin у рад за мање од 5 минута**

Овај водич ће вам помоћи да креирате вашу прву интерактивну визуализацију података користећи Vizualni Admin и српске отворене податке.

## 🚀 Водич за 5 минута

### Предуслови

- Node.js 18+ и npm/yarn
- Основно знање React/TypeScript-а

### Корак 1: Инсталација

```bash
# Клонирајте репозиторијум
git clone https://github.com/acailic/vizualni-admin.git
cd vizualni-admin

# Инсталирајте зависности
yarn install

# Подесите окружење
cp .env.example .env.local
```

### Корак 2: Конфигурисање data.gov.rs API-ја

Додајте конфигурацију data.gov.rs API-ја у `.env.local`:

```env
DATABASE_URL=postgres://postgres:password@localhost:5432/vizualni_admin
DATA_GOV_RS_API_URL=https://data.gov.rs/api/1
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

### Корак 3: Покретање развојног сервера

```bash
# Покрените базу података (ако користите Docker)
docker-compose up -d

# Покрените миграције
yarn db:migrate:dev

# Покрените развојни сервер
yarn dev
```

### Корак 4: Креирајте прву визуализацију

Посетите **http://localhost:3000** и:

1. **Претражите српске скупове података** са data.gov.rs
2. **Изаберите скуп података** (нпр. Популација по општини)
3. **Изаберите тип дијаграма** (стубичасти, линијски, мапа, итд.)
4. **Конфигуришите визуализацију** са бојама и филтерима
5. **Извезите или угнездите** вашу визуализацију

## 🎯 Пробајте овај пример

Ево једноставног примера за визуелизацију српских популационих података:

```typescript
import { DataGovRsClient } from '@acailic/vizualni-admin'
import { BarChart } from '@acailic/vizualni-admin/charts'

// Инициализирајте API клијент
const client = new DataGovRsClient({
  baseUrl: 'https://data.gov.rs/api/1'
})

// Пронађите српске популационе податке
const datasets = await client.searchDatasets({
  q: 'становништво по општинама',
  limit: 10
})

// Креирајте стубичасти дијаграм
const populationChart = new BarChart({
  data: datasets[0].resources[0].data,
  x: 'municipality',
  y: 'population',
  title: 'Популација по општини (2023)',
  colors: ['#3c82f6', '#1d4ed8']
})

// Рендерујте дијаграм
populationChart.render('#chart-container')
```

## 📱 Шта можете креирати

Vizualni Admin подржава више типова визуализација:

### 📊 Дијаграми
- **Стубичасти дијаграми** - Савршени за категоријалне податке
- **Линијски дијаграми** - Идеални за временске серије
- **Областни дијаграми** - Прикажите кумулативне податке током времена
- **Scatter дијаграми** - Корелишу две варијабле
- **Пите дијаграми** - Прикажите пропорције и проценте

### 🗺️ Мапе
- **Choropleth мапе** - Бојом кодирани региони
- **Тачкасте мапе** - Локациони подаци
- **Топлотне мапе** - Визуелизација густине података

### 📋 Табеле
- **Табеле података** - Сортиране и филтрирање
- **Pivot табеле** - Вишедимензионална анализа

## 🇷🇸 Интеграција српских података

Vizualni Admin се безбедно интегрише са [data.gov.rs](https://data.gov.rs):

### Доступне категорије података
- 🏛️ **Влада** - Буџет, запослени, услуге
- 📈 **Економија** - БДП, инфлација, трговина
- 👥 **Демографија** - Популација, миграције, старосне групе
- 🏥 **Здравство** - Болнице, лекари, статистика
- 🎓 **Образовање** - Школе, студенти, перформанси
- 🌍 **Окружење** - Загађење, климат, ресурси

### Пример: Економски индикатори

```typescript
// Претражите економске скупове података
const economicData = await client.searchDatasets({
  organization: 'Републички завод за статистику',
  topic: 'економија',
  limit: 20
})

// Креирајте више-серијски линијски дијаграм
const economicChart = new LineChart({
  data: economicData[0].data,
  series: [
    { key: 'gdp', label: 'БДП (мил. EUR)' },
    { key: 'inflation', label: 'Инфлација (%)' },
    { key: 'unemployment', label: 'Неезапосленост (%)' }
  ],
  timeField: 'year',
  title: 'Економски индикатори - Република Србија'
})
```
