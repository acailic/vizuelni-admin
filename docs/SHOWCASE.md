# Showcase

Welcome to the Vizualni Admin Showcase! This page highlights real-world projects built with Vizualni Admin, demonstrating its versatility in creating beautiful, interactive data visualizations using Serbian open data and beyond. Each project includes descriptions, use cases, links to repositories or live sites, and screenshots.

Projects are categorized by industry. If you've built something with Vizualni Admin, we'd love to feature it here—see the [submission guidelines](#submission-guidelines) below.

## Government

### Serbian Budget Tracker
**Description:** A public dashboard visualizing Serbia's national budget allocations across ministries, built with Vizualni Admin's chart components and integrated with data.gov.rs API for real-time updates.

**Use Case:** Helps citizens understand government spending and promotes transparency in public finance.

**Links:** [GitHub Repository](https://github.com/example/serbian-budget-tracker) | [Live Demo](https://example.com/serbian-budget-tracker)

## Submission Guidelines

We'd love to feature your Vizualni Admin project in our showcase! To submit:

1. **Prepare Your Submission:** Include a brief description, use case, links to repository and live demo, and at least one screenshot (preferably 800x400 or similar aspect ratio).

2. **Categorize Appropriately:** Choose the most relevant industry category (government, NGO, research, journalism). If it doesn't fit, suggest a new category.

3. **Open Source Preference:** Projects should be open source with public repositories for transparency and community benefit.

4. **Quality Standards:** Ensure the project demonstrates best practices, such as responsive design, accessibility, and proper data handling.

5. **Submit via GitHub:** Open an issue on our [GitHub repository](https://github.com/acailic/vizualni-admin) with the title "Showcase Submission: [Project Name]" and include all details in the issue body.

We'll review submissions and add featured projects to this page. Thank you for contributing to the Vizualni Admin community!
