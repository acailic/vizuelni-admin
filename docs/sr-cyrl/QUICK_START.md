# Водич за брзи почетак

> **Quick Start Guide на српском ћирилицом**

Овај водич ће вам помоћи да почнете да користите Визуелни Админ Србије за неколико минута.

---

## Шта вам треба

- ✅ Претраживач (Chrome, Firefox, Safari, или Edge)
- ✅ Интернет веза
- ✅ Подаци у CSV, Excel, или JSON формату (опционо)

---

## Корак 1: Приступите платформи

1. Отворите [vizuelni.rs](https://vizuelni.rs)
2. Кликните на дугме **„Пријави се"**
3. Одаберите начин пријаве:
   - Google налог
   - GitHub налог
   - Е-пошта и лозинка

---

## Корак 2: Креирајте први графикон

### Опција А: Користите пример података

1. Кликните **„Нови графикон"**
2. Одаберите **„Пример података"**
3. Изаберите тип графикона (стубасти, линијски, пита...)
4. Кликните **„Креирај"**

### Опција Б: Отпремите своје податке

1. Кликните **„Нови графикон"**
2. Кликните **„Отпреми податке"**
3. Изаберите CSV или Excel фајл
4. Систем ће аутоматски препознати колоне
5. Изаберите тип графикона
6. Кликните **„Креирај"**

---

## Корак 3: Прилагодите графикон

### Промена наслова

1. Кликните на наслов графикона
2. Унесите нови текст
3. Притисните Enter

### Промена боја

1. Кликните на **„Подешавања"**
2. Одаберите таб **„Боје"**
3. Изаберите палету или дефинишите сопствене боје

### Промена осовина

1. Кликните на осовину коју желите да промените
2. Подесите наслов, минимум, максимум
3. Промене се аутоматски примењују

---

## Корак 4: Сачувајте и делите

### Чување

- Кликните **„Сачувај"** (или Ctrl+S / Cmd+S)
- Одаберите видљивост:
  - **Јавни** - сви могу да виде
  - **Приватни** - само ви можете да видите

### Дељење

1. Кликните **„Дели"**
2. Ископирајте линк
3. Пошаљите линк другима

### Уградња на веб сајт

1. Кликните **„Дели"** → **„Угради"**
2. Ископирајте HTML код
3. Налепите код на свој сајт

---

## Корак 5: Извезите графикон

### PDF

1. Кликните **„Извези"** → **„PDF"**
2. Изаберите величину стране (A4, A3...)
3. Кликните **„Преузми"**

### Слика (PNG/SVG)

1. Кликните **„Извези"** → **„Слика"**
2. Изаберите формат (PNG или SVG)
3. Изаберите резолуцију
4. Кликните **„Преузми"**

---

## Типови графикона

### Стубасти графикон

**Најбоље за:** Поређење вредности између категорија

```
Пример: Буџет по регионима
┌─────────┐
│ ███████ │ Београд
│ ████    │ Нови Сад
│ ██      │ Ниш
│ █       │ Крагујевац
└─────────┘
```

### Линијски графикон

**Најбоље за:** Приказ промена током времена

```
Пример: Популација кроз године
    ┌────────────────────
    │        ┌───┐
    │    ┌───┘   │
    │ ┌──┘       └──
    └────────────────────
      2020  2022  2024
```

### Графикон пита

**Најбоље за:** Приказ удела у целини

```
Пример: Распоред буџета
      ┌───┐
     /     \
    │  45%  │ Образовање
    │       │ 30% Здравство
     \ 25% /  Остало
      └───┘
```

### Географска карта

**Најбоље за:** Приказ података по регионима

```
┌────────────────────┐
│  █████████████     │ ← Војводина
│  ████████████████  │
│      ███████       │ ← Београд
│    ██████████      │
│   █████████████    │ ← Централна Србија
└────────────────────┘
```

## Интеграција са data.gov.rs

### Претрага скупова података

1. Кликните **„Извори података"**
2. Изаберите **„data.gov.rs"**
3. Претражите по кључним речима
4. Кликните на скуп података
5. Систем ће аутоматски учитати податке

### Доступни скупови података

- Буџет Републике Србије
- Попис становништва
- Економски показатељи
- Административна подела
- И још 1.200+ скупова

## Честа питања

### П: Да ли је платформа бесплатна?

**О:** Да, основна верзија је потпуно бесплатна. Напредне функције су доступне у enterprise верзији.

### П: Да ли могу да користим ћирилицу?

**О:** Да, платформа у потпуности подржава српску ћирилицу и латиницу.

### П: Где се налазе моји подаци?

**О:** Подаци се чувају на сигурним серверима у Србији/ЕУ.

### П: Да ли могу да делом приватне графиконе?

**О:** Да, можете подесити видљивост на „Приватни" за сваки графикон.

### П: Како да пријавим проблем?

**О:** Пошаљите е-пошту на opendata@ite.gov.rs
