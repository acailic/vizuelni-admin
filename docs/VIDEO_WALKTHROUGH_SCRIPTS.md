#### data.gov.rs Integration (5:00-7:00)

**Visual**: API integration code

**Narrator**:

> "За директно учитавање података са data.gov.rs, користите наш
> конектор."

```typescript
import { DataGovRsConnector } from '@vizualni/connectors';

async function loadBudgetData() {
  const connector = new DataGovRsConnector();

  // Search for datasets
  const results = await connector.search('буџет');

  // Load specific dataset
  const dataset = await connector.getDataset('budget-ministry-2024');

  // Convert to chart-ready format
  const chartData = connector.toChartData(dataset, {
    x: 'ministry',
    y: 'amount',
    filter: { year: 2024 },
  });

  return chartData;
}
```

**Error handling**:

```typescript
try {
  const data = await loadBudgetData();
} catch (error) {
  if (error instanceof DataGovRsError) {
    console.error('API error:', error.message);
  }
}
```
