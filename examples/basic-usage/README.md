# Basic Usage Example

This example demonstrates how to use `@vizualni/react` charts in a React
application.

## Quick Start

```bash
npm install
npm start
```

## Try it on CodeSandbox

[Open in CodeSandbox](https://codesandbox.io/s/github/acailic/vizualni-admin/tree/main/examples/basic-usage)

## What's Included

- LineChart with date-based data
- BarChart with categorical data
- PieChart with labeled segments
