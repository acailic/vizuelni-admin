Za programere

Ugradite vizuelizacije u svoje aplikacije sa nekoliko redova koda.

population-chart.tsx
import { BarChart } from '@vizualni/react';

const data = [
  { region: 'Београд', population: 1688667 },
  { region: 'Војводина', population: 1736961 },
  { region: 'Шумадија и Западна Србија', population: 1949764 },
  { region: 'Јужна и Источна Србија', population: 1507052 },
];

const config = {
  type: 'bar',
  x: { field: 'region', type: 'string' },
  y: { field: 'population', type: 'number' },
} as const;

export function PopulationChart() {
  return (
    <BarChart
      data={data}
      config={config}
      width={720}
      height={420}
    />
  );
}

Platforma je izgrađena na standardnim veb tehnologijama. Prilagodite konfiguracije grafikona, integrišite sa svojim procesom podataka ili doprinesite kodnoj bazi otvorenog koda.
