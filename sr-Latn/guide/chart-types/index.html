Типови графикона

Стубасти и тракасти графикони

Упоређивање вредности између категорија

Користити када

  • •Рангирање категорија
  • •Упоређивање по регионима
  • •Приказ одступања од циља
  • •Мање од 10 категорија

Избегавати када

  • •Више од 15 категорија
  • •Трендови током времена
  • •Пропорције (користити питу)
  • •Негативне вредности за тракасте

Примери из јавних података

Популација по градовимаБолнички кревети по окрузима

Честе грешке

Превише категорија отежава читање; користи хоризонталне тракасте за дугачке ознаке

Приступачност:Добра подршка за читаче екрана; табела алтернатива доступна
Алтернатива: Табела за прецизне вредности

Линијски графикони

Трендови и промене током времена

Користити када

  • •Временске серије
  • •Упоређивање више серија
  • •Истицање образаца и трендова
  • •Непрекидне вредности

Избегавати када

  • •Категоријалне податке
  • •Више од 5 серија (претрпано)
  • •Мали број тачака (користи тачкасти)
  • •Негативне вредности без нуле

Примери из јавних података

Квартални БДПМесечна стопа инфлације

Честе грешке

Не укључуј нулу на Y-оси када је битна; превише линија отежава разликовање

Приступачност:Тренд информације у aria-describedby; подршка за тастатуру (стрелице)
Алтернатива: Површински за наглашавање волумена

Пита и прстенасти графикони

Пропорције и састав целине

Користити када

  • •Састав где збир = 100%
  • •3-7 категорија
  • •Једна серија података
  • •Брзи увид у пропорције

Избегавати када

  • •Више од 7 категорија
  • •Упоређивање више серија
  • •Сличне вредности (тешко разликовати)
  • •Прецизне вредности (користи табелу)

Примери из јавних података

БДП по секторима економијеСтуденти по нивоима образовања

Честе грешке

Превише сегмената; не ређај по величини ако није логично; избегавај 3D ефекте

Приступачност:Ограничена подршка; ОБАВЕЗНО обезбедити табелу алтернативу
Алтернатива: Тракасти за поређење пропорција

Површински графикони

Кумулативне вредности и састав током времена

Користити када

  • •Волумен током времена
  • •Слоежни састав (stacked)
  • •Део-целина трендови
  • •Наглашавање магнитуда

Избегавати када

  • •Прецизно читање вредности
  • •Више од 4 слоја
  • •Преклапајуће серије без транспарентности
  • •Негативне вредности

Примери из јавних података

Извоз по секторима (годишње)Потрошња енергије по извору

Честе грешке

Редослед слојева утиче на читање; највећа категорија иде на дно

Приступачност:Слојеви се тешко разликују; табела алтернатива препоручљива
Алтернатива: Линијски за чистије поређење

Графикони расејања

Корелација између две варијабле

Користити када

  • •Истраживање односа варијабли
  • •Приказ кластера и изузетака
  • •Велики број тачака
  • •Хипотезе о корелацији

Избегавати када

  • •Мали број тачака (користи тачкасти)
  • •Временске серије
  • •Категоријалне податке
  • •Без јасне корелације

Примери из јавних података

Здравствена потрошња vs очекивани животБДП по глави становника vs стопа писмености

Честе грешке

Не подразумевај узрочност; укључи тренд линију ако је релевантна

Приступачност:Тешко за читаче екрана; табела са вредностима обавезна
Алтернатива: Мехурићи за трећу димензију

Географске карте

Географска дистрибуција података

Користити када

  • •Просторни обрасци
  • •Упоређивање региона
  • •Локалне варијације
  • •Јавни сектор и демографија

Избегавати када

  • •Мале географске разлике
  • •Више од 7 боја
  • •Без контекста (додај легенду)
  • •Временске серије (користи анимацију)

Примери из јавних података

Густина насељености по окрузимаПокривеност вакцинацијом по општинама

Честе грешке

Избегавај црвено-зелене палете; користи приступачне палете; додај шаблоне за далтонисте

Приступачност:Обезбеди табелу алтернативу; ARIA ознаке за регионе; контраст боја ≥3:1
Алтернатива: Тракасти за прецизно поређење

Табеле

Прецизне вредности и приступачност

Користити када

  • •Тачне вредности
  • •Приступачност за читаче екрана
  • •Преглед за преузимање
  • •Више мера по категорији

Избегавати када

  • •Визуелни трендови
  • •Велике табеле без пагинације
  • •Превише колона
  • •Брзи увид у обрасце

Примери из јавних података

Статистички годишњакБуџетска извршења по министарствима

Честе грешке

Без заглавља колона; нема сортирања; лош контраст; претрпани подаци

Приступачност:Најбоља подршка; користи scope атрибуте; заглавља за сваку колону
Алтернатива: Стубасти за визуелни приказ
